# Claret

A warm, refined dark and light color theme for VS Code.

Ported from [claret.nvim](https://github.com/cfbender/claret.nvim).

## Variants

- **Claret Dark** — Deep burgundy-tinted dark theme
- **Claret Light** — Warm parchment light theme

## Install

Search "Claret" in the VS Code Extensions sidebar, or:

```
ext install cfbender.claret
```

## Palette

Built on a locked palette of rose, gold, sage, slate, and terra tones. Keywords use rose, functions use gold, strings use sage, types use slate.
